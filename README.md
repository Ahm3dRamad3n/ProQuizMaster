# 🏆 Pro Quiz Master - نظام المسابقات التفاعلي

## ✨ المميزات الاحترافية (Key Features)

  * **⚡️ سرعة استجابة فائقة:** تحديثات لحظية بفضل تقنية الـ Real-time في Firebase.
  * **🎨 واجهة Glassmorphism:** تصميم عصري وجذاب باستخدام Tailwind CSS مع أنيميشن سلس.
  * **🔒 نظام حماية:** منع دخول أي متسابق جديد بعد اكتمال العدد أو بمجرد بدء أول سؤال.
  * **🎮 تحكم كامل (Admin):** لوحة تحكم مخصصة لإرسال الأسئلة، توزيع النقاط (+/-)، وتصفير الـ Buzzer.
  * **🥇 لوحة المتصدرين:** جدول نقاط حي يتحدث تلقائياً ليظهر ترتيب المتسابقين أمام الجميع.
  * **🎉 شاشة التتويج:** عرض احتفالي خاص بالمركز الأول عند انتهاء المسابقة.
